Artifact Enhancements

1. Software Engineering and Design

Milestone Two Narrative (Word)

Refactored the backend Express API for modularity, maintainability, and cleaner architecture. Integrated middleware, centralized logging, and updated routes.

2. Algorithms and Data Structures

Milestone Three Narrative (Word)

Enhanced sorting and time-based logic in the frontend, optimized utilities for formatting and comparison, and removed redundant control structures.

3. Databases

Milestone Four Narrative (Word)

Improved authentication with JWT, redesigned Mongoose schemas, and optimized MongoDB queries with better indexing and validation handling.

Artifact Overview: Travlr Getaways

  • Frontend: Angular
  • Backend: Node.js + Express
  • Database: MongoDB via Mongoose
  • Authentication: JSON Web Tokens (JWT)
  • Extras: Handlebars views, Winston logging, Bootstrap styling
